Để tạo một trang web GitHub Pages có địa chỉ dạng https://tennguoidung.github.io, bạn có thể làm theo các bước sau:

Bước 1: Đăng nhập GitHub

Truy cập: GitHub

Đăng nhập vào tài khoản của bạn.

Bước 2: Tạo repository mới

  1. Nhấn New (hoặc dấu + → New repository).
  2. Đặt tên repository theo đúng cú pháp:
tennguoidung.github.io

Ví dụ:

  • Tên tài khoản GitHub: nguyenvana
  • Tên repository phải là:
nguyenvana.github.io
  • Chọn Public.
  • Có thể tích chọn Add a README file.
  • Nhấn Create repository.

Bước 3: Thêm trang web

Trong repository vừa tạo:

  1. Chọn Add file → Create new file.
  2. Đặt tên file:
index.html

Nhập nội dung ví dụ:

<!DOCTYPE html>
<html>
<head>
    <meta charset="UTF-8">
    <title>Trang GitHub Pages đầu tiên</title>
</head>
<body>
    <h1>Xin chào GitHub Pages!</h1>
    <p>Đây là website đầu tiên của tôi.</p>
</body>
</html>

Nhấn Commit changes.

Bước 4: Bật GitHub Pages

  1. Vào Settings của repository.
  2. Chọn Pages ở menu bên trái.
  3. Trong mục Build and deployment:
    • Source: chọn Deploy from a branch.
    • Branch:
      • Chọn main
      • Folder: / (root)
  4. Nhấn Save.

Bước 5: Chờ website được triển khai

Sau khoảng 1–5 phút, GitHub sẽ tạo website.

Địa chỉ sẽ là:

https://tennguoidung.github.io

Ví dụ:

https://nguyenvana.github.io

Bước 6: Cập nhật website

Mỗi lần muốn chỉnh sửa:

  • Sửa file index.html hoặc thêm các file như:
    • style.css
    • script.js
    • thư mục images/
  • Commit các thay đổi.
  • GitHub Pages sẽ tự động cập nhật sau khoảng 1–2 phút.
